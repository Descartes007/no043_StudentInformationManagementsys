## 项目介绍

基于JSP+Servlet实现的学生信息管理系统，主要功能如下：

### 【管理员】
学生信息管理：管理员可以添加、编辑和删除学号，姓名，性别，年龄，籍贯，电话，班级

### 项目技术
编程语言：Java
数据库：MySQL
前端技术：JSP、JavaScript、bootstrap、JQuery
后端技术：Servlet、JDBC

## 环境
